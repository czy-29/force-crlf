# force-crlf
Force a file to use CRLF line breaks.

```
cargo install force-crlf
```

And then:
```
force-crlf relative_path/hello.rs
```
Or use absolute path:
```
force-crlf D:\absoulute_path\hello.rs
```
to make hello.rs CRLF.
